Lou-extensions

Collection of abandoned extensions for the browser game Lord Of Ultima

View project onGitHub

This project aims to provide a community driven maintenance for Chrome extensions and GreaseMonkey scripts for LoU that are not actively maintained by their original authors anymore but still add vital functionality to LOU. On a regular basis EA and Phenomic studios launch new patches that break these extensions so they need to be fixed.

Try a search for LoU BoS on userscripts.org and you'll see plenty of customized versions fixing various issues. I'm offering an alternative by hosting the code on GitHub where anyone can directly push fixes directly in the source code.

Install latest lou-extensions


Google Chrome version 18 or higher:
install lou-extension version 0.4.0 for Chrome (release date 17th July 2012)


Greasemonkey 0.9.16 or higher:
install lou-extension version 0.4.0 for Greasemonkey (release date 17th July 2012)
