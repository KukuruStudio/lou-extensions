Lou-extensions

Collection of abandoned extensions for the browser game Lord Of Ultima

View project on GitHub

This project aims to provide a community driven maintenance for Chrome extensions and GreaseMonkey scripts for Lord Of Ultima (LoU) that are not actively maintained by their original authors anymore but still add vital functionality to LOU. On a regular basis EA and Phenomic studios launch new patches that break these extensions so they need to be fixed.

Install latest lou-extensions

This version takes advantage of the auto-update features that are built-in in recent versions of Greasemonkey and Google Chrome. View the change log on the project wiki.


Google Chrome version 18 or higher:
install lou-extension version 0.5.0 for Chrome (release date 24th July 2012)


Greasemonkey 0.9.16 or higher:
install lou-extension version 0.5.1 for Greasemonkey (release date 24th July 2012)

Older versions

Extension installed from this section do not use auto-update features. Install from here if you don't want to follow the automatic update cycle. This could be useful in case a newer version doesn't work

install lou-extension version 0.4.0 for Chrome (release date 17th July 2012)
install lou-extension version 0.4.0 for Greasemonkey (release date 17th July 2012)
